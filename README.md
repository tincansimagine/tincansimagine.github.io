**깡갤 번역기 v1.9.0**

해당 번역기는 디씨인사이드 '[깡갤 비공갤 대피소](https://gall.dcinside.com/mini/tincans)' 익명 미니갤을 제외한 그 어느 곳에도 원본에 대한 언급/배포 및 2차 저작물 배포를 허용하지 않습니다. 깡갤 대피소 미니갤 내에서 명시적으로 동일한 CCL (CC BY-NC-ND) 및 같은 조건을 적용 + 원저작물에 대한 출처 표기하는 경우에 한하여, 원본에 대한 수정 버전 배포 가능합니다. 만약 해당 번역기가 명시된 사이트 이외의 곳에서 발견된다면 제작자 본인이 절대 아니기 때문에 댓글로 알려주거나 공익제보 바랍니다. 어떤 이유에서든 상관없이 해당 번역기의 '상업적 이용'은 절대 금지합니다.

---

## 개요

정적 페이지(`GitHub Pages`)에서 동작하는 번역 UI입니다. **Google Gemini**, **OpenAI**, **Anthropic**, **Cohere**, **Z.AI GLM**, **Vertex AI** 등을 선택해 번역할 수 있고, 리버스 프록시(OpenAI 호환)도 설정할 수 있습니다.

## 주요 기능

- **텍스트 번역**: 한↔영 방향, 커스텀 프롬프트·템플릿, 용어집(카테고리), 단어 규칙, 히스토리·배치 번역
- **파일 불러오기**: TXT, PDF, DOC/DOCX (원문 칸으로 추출)
- **URL 본문 가져오기**: 페이지 URL을 넣고 「본문 가져오기」— 브라우저 CORS를 피하기 위해 **[Jina Reader](https://r.jina.ai/)** 로 본문을 받아옵니다. 로그인·봇 차단·동적 페이지 등에서는 실패하거나 내용이 비어 있을 수 있습니다.
- **이미지 번역(비전)**: 원문 도구줄의 이미지 아이콘으로 사진을 첨부하면, OCR·번역을 한 번에 요청할 수 있습니다.  
  지원 판정 모델 예: **Gemini / Vertex의 Gemini**, **GPT-4o·GPT-5 계열**, **Claude 3·4** 등. (커스텀 모델은 공급자가 `gemini`, `vertex`, `openai`, `anthropic`일 때 버튼이 표시됩니다.)
- **마크다운 미리보기**: 결과 패널에서 편집 ↔ 미리보기 전환
- **테마·폰트**: 다중 테마, `translator-fonts.css` 및 Google Fonts(Noto Sans KR 등) 연동

## 기술 스택

- HTML/CSS/JS (`index.html`, `tin-translator.css`, `translator-fonts.css`, `translator.js`)
- Marked(마크다운), PDF.js, Mammoth(DOCX)

## 배포

저장소를 GitHub Pages 루트로 두고 `index.html`이 열리도록 설정하면 됩니다. API 키는 브라우저 `localStorage`에 저장되므로 **공용 PC에서는 주의**하세요.
