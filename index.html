단축키 목록

  • Ctrl + Enter : 번역하기
  • Ctrl + S : 프롬프트 저장
  • Esc : 번역 취소
  • Ctrl + I : 아보카도 모드 토글
  • Ctrl + L : 라이트 모드 토글
  • Ctrl + D : 다크모드 토글
  • Alt + 1 : 한→영 전환
  • Alt + 2 : 영→한 전환
  • Ctrl + E : 입력 텍스트 전체 삭제 (단, 입력 텍스트 창이 눌린 상태에서만 동작)
